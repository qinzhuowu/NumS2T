NumS2T

Math Word Problem Solving with Explicit Numerical Values

## Requirement

- Python2.7
- Pytorch 1.1.0
- numpy
- nltk

# Train the model.
python run_seq2tree.py

# Evaluate the model.
python evaluate.py
